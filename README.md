<h2> 🍸 Authentication Service </h2>

```bash
# go to authentication-svc
$ cd authentication-svc

# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
